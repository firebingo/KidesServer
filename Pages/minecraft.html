Instructions:

Go to the forge page, expand show all downloads, and download either the installer or installer-win for 12.18.3.2272.

This should create a profile in the Minecraft launcher named something like 1.10.2-forge1.10.2-12.18.3.2272.

Go to the Launch Options/Edit Profile for this profile and ensure that the version selected looks correct.

Launch the game using this profile once then close it.

Download the zip from the top of this page and extract its contents to %appdata%/.minecraft/mods

If you have used mods before make sure to clear out this directory first.

You should now be able to launch the game. Message either me (Firebingo) or KiD so that we can whitelist you on the server. Then you can join the server.
